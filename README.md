## Getting Started <a name = "getting_started"></a>

### Installing

First, rename .env.bak to .env and set the config as you want

```
PORT=5000
HOSTNAME=localhost
ID_LENGTH=8
```

And, you must install required dependencies by command

```
npm install
```

After that, run command

```
npm run dev
```

Or, you can run command

```
npm run build
npm run start
```
